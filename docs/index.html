About This Project

PA-Gladdy is a specialized backport and optimization of the popular Gladdy arena addon, tailored specifically for Project Ascension (Classless WoW).

While originally built for WotLK 3.3.5a, Ascension's unique server architecture (custom spells, modified APIs, data delays) breaks standard 3.3.5 addons. This version includes 12 rounds of critical hotfixes, a custom compatibility layer (replacing !!!ClassicAPI), and optimizations for Ascension's infinite class possibilities.

Ascension Native

Polyfilled APIs (C_Timer, RegisterUnitEvent) and fixes for Ascension-specific data delays ensure smooth operation.

Arena Ready

Fixes for frame duplication, health bar desyncs, and cast bar tracking make it purely competitive-grade.

Taint Free

Carefully patched to avoid tainting Blizzard's secure environment. Macros, spellbook, and action bars work perfectly.

Optimized Core

Removed bloat and heavy dependencies. Includes custom GladdyCompat layer for maximum performance.
