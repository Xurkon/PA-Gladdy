## üì• Installation

1. Download the latest release.
2. Extract the `Gladdy` folder.
3. Place it in `Ascension Launcher\resources\client\Interface\AddOns\`.
4. **IMPORTANT**: Remove any `!!!ClassicAPI` folders if present (Gladdy no longer needs it).

## üîß Usage

- **/gladdy** - Open the configuration menu.
